##sizeof

sizeof可以用来计算一个变量或者一个常量、一种数据类型所占的内存字节数。

sizeof(常量/变量/数据类型)

##数组

int a[10];
int  a[5];   // 整型常量
int  b['A'];  // 字符常量，其实就是65
int  c[3*4];  // 整型常量表达式

数组内存地址随着 a[0] 依次按 数据类型 所占字节个数递增。

##字符串

 char a[3] = {'m', 'j', '\0'};
 
 其实特殊的字符数组。
 普通的字符数组
 char a[2] = {'m','j'};  '\0' 是一个 ASCII 码值为0的字符.表示什么也不干。
 
 %s  输出字符串对应标识符。
 %x  输出地址对应标识符
 
 **puts函数:** 其会从字符串数组的首个下标开始输出字符.
 **scanf:** 接收用户输入字符、拼接成字符串数组。
 **gets:** 和scanf 一致, 结束之后系统会自动追加 \0。
